# Power BI Intro

Power BI初心者向けのHTMLスライドです。

ExcelやCSVなどのデータをPower BI Desktopに取り込み、整形し、レポートを作成して共有するまでの流れを、操作イメージ中心で学べる内容にしています。

## 対象者

- Excel集計からPower BIに進みたい人
- 基本的なIT知識がある初心者
- Power BIでどのようなレポートが作れるか知りたい人

## 内容

- Power BIとは
- Power BIでできること
- データ取得
- Power Queryによるデータ整形
- Power BI Desktopの3つのビュー
- レポートビューでのレポート作成
- DAX式と指標
- 完成レポートのイメージ
- Power BI Serviceへの発行と共有
- ライセンスの考え方

## 使い方

`index.html` をブラウザで開くとスライドを表示できます。

- 次へ: 右矢印キー / スペースキー / 右下の次ボタン
- 前へ: 左矢印キー / 右下の前ボタン
